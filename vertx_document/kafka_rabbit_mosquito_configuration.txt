download kafka from https://kafka.apache.org/downloads and unzip it in c drive

how to run kafka zookeeper
C:\kafka_2.12-2.6.0\kafka_2.12-2.6.0\bin\windows>zookeeper-server-start.bat zookeeper.properties

how to run kafka server
C:\kafka_2.12-2.6.0\kafka_2.12-2.6.0\bin\windows>kafka-server-start.bat server.properties

how to create kafka topic

kafka-topics --create --topic test-topic1 --bootstrap-server localhost:9092 --replication-factor 1 --partitions 3

kafka-topics --list --bootstrap-server localhost:9092
